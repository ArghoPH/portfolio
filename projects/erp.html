Shakti Digital Loan Process Design

Designing a seamless, user-centered experience for applying and managing loans digitally.

Project Overview

This project focuses on creating a digital loan application interface that is intuitive, responsive, and easy for users to navigate. I designed the complete user flow and screens in Figma, ensuring clarity at every step.

The digital loan process guides users from onboarding to loan submission in a smooth, step-by-step manner. Each screen was carefully crafted to reduce friction and improve usability. The user flow maps out interactions like data entry, verification, and confirmation, making the process straightforward and user-friendly.

Technologies Used
